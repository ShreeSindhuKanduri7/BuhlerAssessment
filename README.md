# Saucedemo Test Automation

## ✅ Setup
```bash
npm install
```

## 🚀 Run Tests
```bash
npx playwright test
```

## 📊 View Test Report
```bash
npx playwright show-report
```

## 📁 Structure
```
saucedemo-tests/
├── tests/           # Test specs
├── pages/           # Page Object Model
├── test-data/       # Test data in JSON
├── playwright.config.ts
├── package.json
├── README.md
```

## 🔧 Tech Stack
- [Playwright](https://playwright.dev/)
- TypeScript
- Page Object Model
- Data-driven tests
